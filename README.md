# Malaysian Road Accident Severity Analysis and Prediction

## Overview
This project analyzes Malaysian road accident trends from 2000 to 2016 using 
historical data from MIROS and DOSM. It applies exploratory data analysis and 
regression modelling to identify key factors influencing road fatalities and 
forecast future trends.

## Course
WIE2003 Introduction to Data Science — Group 1, OCC 1

## Project Structure
- `data/` — Cleaned dataset
- `notebooks/01_eda.ipynb` — Exploratory Data Analysis
- `notebooks/02_model.ipynb` — Regression Modelling
- `app/app.py` — Streamlit Dashboard

## Setup Instructions
1. Clone this repository
2. Run `pip install -r requirements.txt`
3. To launch the dashboard: `streamlit run app/app.py`
